Changelog

Dreams & Machines Character Creator. Short timeline of user-facing changes.

Version 1.10

  • Added tracked uses for Haggler, Just What They Need, I Know a Guy, Ambush, and Tough, each with a Use control and a used state.
  • Added New Session and New Adventure controls beside End Scene, so abilities limited to a session or an adventure refresh at the right boundary.
  • Moved Tech Level up to the name line and stacked the Origin, Archetype, and Temperament tags, giving the Goals boxes more room.
  • Origin, Archetype, and Temperament tooltips now show what each one is, instead of repeating its own label.
  • Expanded panels no longer snap shut and the page no longer jumps when you equip an item, take a rest, or confirm an action.
  • Fixed the version link in the header, which was showing an older release than the one you were using.

v.1.09

  • Added expandable reference cards for all core actions, including Counterattack.
  • Added confirmed Momentum spending for Second Wind, Create Truth, Ask a Question, Damage, Reduce Time, Follow-Up, and Counterattack.
  • Added once-per-scene Adrenaline Rush controls and an End Scene reset.
  • Added an interactive Spear's Blend scene effect and automatic Bold and Spear's Blend Spirit bonuses.
  • Added Breather, Break, and Bed controls which restore Spirit and recharge eligible Powered item stacks.
  • Added Charged and Discharged state tracking for Powered items and weapons, including manual handling for Powered (Special).
  • Added Custom Items and Knowledge Fragments as persistent free-text lists.
  • Fixed starting Spirit and Supply Point initialization and added a confirmed Supply Point reset.
  • Relabeled Character Truths to Truths, added Truth type guidance, and connected action controls to Truth creation.
  • Reorganized the finished sheet and refined the identity header, resource layout, and play controls.
  • Adopted the stable app filename dnm-character-creator.html.

v.1.08

  • Added equipped state and structured equipped-item effects.
  • Added fuller catalogue descriptions, effect notes, and improved equipment tooltips.
  • Added editable current resource values on the finished sheet.
  • Added editable Truth and Injury tracking.
  • Improved catalogue scrolling, floating tooltips, item details, and printable sheet behavior.

v.1.07

  • Added portrait web addresses, live portrait preview, and portrait rendering on the finished sheet.
  • Persisted portrait addresses through share codes, imports, and local saves.
  • Expanded Supportive and Rivalry Bond rules tooltips.
  • Improved Owned Item disclosure, spacing, and expanded content.
  • Fixed Item Catalogue search focus loss while typing.

v.1.06

  • Added a clickable changelog link from the version label in the app header.
  • Added this standalone changelog page with a Back to Character Creator link.
  • Moved Growth to the second-to-last section of the character sheet.
  • Moved Character Share Code to the bottom of the character sheet.
  • Fixed Item Catalogue sub-filter rendering so sub-filters appear after selecting a main filter.

v.1.05

  • Exhaustion now disables rolls using the affected attribute.
  • Exhausted attributes are struck through in the Dice Roller dropdown.
  • Item Catalogue sub-filters now appear after selecting a main category.
  • Character Share Code moved to second-to-last section.
  • Growth moved to the bottom of the sheet.

v.1.04

  • Added local saved characters in browser local storage.
  • Added finalized character mode.
  • Moved Character Share Code to a collapsible bottom section.
  • Added dynamic Spirit, Supply Points, Coin, Growth, and Momentum fields.
  • Expanded equipment, item, vehicle, GLIF, and Bond tooltips.
  • Made Item Catalogue and Starting Equipment collapsible.
  • Added clickable Exhaustion States.
  • Made inventory items expandable.

v.1.03

  • Added full skill names on the sheet.
  • Added attribute and skill tooltips.
  • Added the full item catalogue.
  • Added Exhaustion States section.
  • Added Dice Roller.
  • Added shared roll room support.
  • Added collapsible share code.
  • Fixed character-code base64 import reliability.
